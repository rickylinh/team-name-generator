# Team Name Generator - 团队名称生成器

## 项目简介

这是一个基于 Vue 3 + Vite 构建的现代化团队名称生成器 Web 应用。参考了 [name-generator.org.uk](https://www.name-generator.org.uk/team/) 网站的功能设计，采用现代简约风格，提供简洁优雅的用户体验。

## 功能特性

### 核心功能
- ✅ **自定义生成**: 支持通过颜色、形容词、地点、动物、人群等选项自定义生成团队名称
- ✅ **随机生成**: 一键随机生成团队名称，无需输入任何内容
- ✅ **智能组合**: 基于中文语言习惯，智能组合多个输入项生成自然流畅的团队名称
- ✅ **名称展示**: 优雅的名称展示界面，支持重新生成

### 附加功能
- ✅ **社交分享**: 支持分享到 Twitter、Facebook、Reddit 等社交平台
- ✅ **链接复制**: 一键复制分享链接
- ✅ **邮件订阅**: 可选的邮件订阅功能（前端实现，需连接后端）
- ✅ **响应式设计**: 完美适配桌面端和移动端

### 设计特色
- 🎨 **现代简约风格**: 干净的白色布局，优雅的圆角卡片设计
- 🎨 **微妙阴影效果**: 精致的阴影提升视觉层次
- 🎨 **柔和强调色**: 使用柔和的紫色和粉色作为强调色
- 🎨 **优雅字体**: 使用系统默认的无衬线字体，确保清晰易读
- 🎨 **流畅动画**: 平滑的过渡动画和交互效果
- 🎨 **响应式网格**: 自适应的网格布局，完美适配各种屏幕尺寸

## 技术栈

- **框架**: Vue 3 (Composition API)

- **特性**: 纯前端应用，无需后端服务

## 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0 或 yarn >= 1.22.0

### 安装步骤

1. **安装依赖**
   ```bash
   npm install
   # 或
   yarn install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   # 或
   yarn dev
   ```

   开发服务器将在 `http://localhost:3000` 启动，浏览器会自动打开。

3. **构建生产版本**
   ```bash
   npm run build
   # 或
   yarn build
   ```

   构建产物将输出到 `dist` 目录。

4. **预览生产构建**
   ```bash
   npm run preview
   # 或
   yarn preview
   ```

## 使用说明

### 生成团队名称

1. **自定义生成**:
   - 在表单中填写一个或多个选项（颜色、形容词、地点、动物、人群）
   - 点击"生成名称"按钮
   - 系统会根据您的输入智能组合生成团队名称

2. **随机生成**:
   - 点击"🎲 随机生成"按钮
   - 系统会从预设词汇库中随机组合生成团队名称

### 分享功能

- 生成名称后，可以使用底部的分享按钮分享到社交平台
- 点击"复制链接"可以复制当前页面链接

### 邮件订阅

- 在邮件订阅区域输入您的邮箱地址（可选）
- 点击"订阅"按钮（当前为前端演示，实际需要连接后端API）

## 核心功能实现

### 名称生成算法

名称生成器支持两种模式：

1. **基于用户输入生成**:
   - 收集用户输入的非空字段
   - 按照中文语言习惯组合（使用"的"连接）
   - 智能添加"队"后缀

2. **随机生成**:
   - 从预设词汇库中随机选择 2-3 个不同类型的词汇
   - 按照相同规则组合生成

### 词汇库

内置丰富的词汇库，包括：
- 颜色：16 种常用颜色
- 形容词：24 个描述性词汇
- 地点：24 个地点名称
- 动物：24 种动物名称
- 人群：24 种人群类型
